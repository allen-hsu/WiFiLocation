# WiFiLocation
### wifi室内定位与人数统计项目
这个项目主要实现通过手机端app根据wifi信号强度的实现室内定位。

#### **app**
语言：java (android)
环境：windows 10
工具：Android Studio
主要功能：实现wifi信号强度的离线采集， 在线定位，步长计算，步数统计，步行轨迹推算。



#### **server**
语言：c#(.net)
环境：windows 10
工具：vs2013
服务：IIS
主要功能：实现与app交互的接口：定位接口，采集数据接口。实现定位算法等核心代码。实现定位精度统计的展示页面。可以展示不同算法定位的结果和误差等。
